# Fiit Advisory — Website (English)

Static website for **Fiit Advisory**, Amsterdam.

## Structure

This is a **single-file site** — all HTML, CSS, and JS live in `index.html`.
No build step, no dependencies to install, no path issues.

## Deploy via GitHub Actions

1. Push to GitHub (`git push -u origin main`)
2. Go to **Settings → Pages → Source → GitHub Actions**

Site goes live at `https://YOUR_USERNAME.github.io/YOUR_REPO/`

Every push to `main` triggers an automatic redeploy.
